## Instalação

Instalação rápida: 
```bash
pip install -r requirements.txt
```

Sugestão  
Crie um ambiente virtual antes de instalar:
```bash
python -m venv .venv
source .venv/bin/activate   # Linux macOS
.venv\Scripts\activate      # Windows
pip install -r requirements.txt

```
